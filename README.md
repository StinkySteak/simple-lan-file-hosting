# Simple LAN File hosting using Node express

### Prerequisites
1. Node Js Installed

### How to use?

1. run `init.bat` in the repository
2. Create a folder named `public` in the folder
3. Place any file you want to host
4. run `run_server.bat`
5. Access it by going to browser on address http://{localip}/{yourFileName.png} (on your LAN device)

localip can be gained from your `IPv4 Address` on your active internet adapter, they should form similar to this: `192.168.xxx.xxx`
